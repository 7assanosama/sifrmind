# Changelog

## [0.1.0] — 2026-07-13

### Added

- Next.js 16 App Router with internationalized routing (`next-intl`)
- Arabic (RTL) / English (LTR) full localization
- Apple-inspired landing page with interactive Canvas particle background
- Sections: Hero, Why Sifr, Products, Principles, Features, Process, FAQ, CTA
- Contact page with email, WhatsApp, Instagram, and Facebook contact cards
- Legal pages: Terms of Service, Privacy Policy
  - Sticky table of contents, anchor links, reading progress, back-to-top
- Glassmorphism UI components: GlassCard, Button, Badge, Container, Section
- Scroll-triggered reveal animations with reduced-motion support
- Monochrome dark theme with CSS custom properties
- Responsive navigation with mobile slide-out menu
- SEO metadata, Open Graph, Twitter cards, JSON-LD breadcrumbs
- Print styles and accessibility skip-to-content link

### Technical

- TypeScript strict mode
- Tailwind CSS 4 with `@theme` design tokens
- Framer Motion for animations
- ESLint with `eslint-config-next` (core-web-vitals + typescript)
- Turbopack for development and production builds
